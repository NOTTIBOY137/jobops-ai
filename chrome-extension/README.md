# JobOps AI Chrome Extension

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select the `chrome-extension` folder

## Configuration

1. Click the extension icon in your browser toolbar
2. Enter your JobOps AI API URL (default: http://localhost:3000)
3. Click "Save Settings"

## Usage

1. Navigate to a job posting on:
   - LinkedIn Jobs
   - Indeed
   - Greenhouse
   - Lever
   - Workday
   - Ashby
   - SmartRecruiters
   - Or any job board

2. Click the "Save to JobOps" button that appears on the page
3. The job will be saved to your JobOps AI dashboard

## Features

- Automatically detects job details (company, role, location, description)
- Identifies ATS type (Greenhouse, Lever, etc.)
- One-click save to your JobOps pipeline
- Works on major job boards
